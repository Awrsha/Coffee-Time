## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Awrsha/Coffee-Time.git

# Navigate to project directory
cd Coffee-Time

# Install dependencies
pip install -r requirements.txt

# Setup environment variables
cp .env.example .env
```
